　　萧逸曾听过一个传言。
<br/>　　金光险地，之所以称为金光险地，便是因为这里的夜晚，会忽然出现氤氲金光，笼罩整片辽阔森林。
<br/>　　而同时，这里是万金地域内的险地。
<br/>　　万金府，又称作金光府。
<br/>　　故此险地，有此称呼。
<br/>　　但，在这其中，还有一个更为古老的传言。
<br/>　　千万年前，这里，曾是一位上古强者的修炼之地。
<br/>　　这位上古强者修炼之时，哪怕是单单盘膝而坐，清修感悟，所散发的武道气息，便足以震慑千万里。
<br/>　　千万里范围内，万物无敢靠近者。
<br/>　　每到入夜，这位上古强者身上，便光芒冲天。
<br/>　　万丈金光，笼罩四周，仿佛能与天地比肩。
<br/>　　时至不知多少年后，这位上古强者离开此地，创立了属于他的家族。
<br/>　　而此地，历经多年，仍旧有金光残留。
<br/>　　每到入夜，便金光飘荡，亘古不散。
<br/>　　不错，这位强者，正是创立万金府的那位上古强者。
<br/>　　这些飘荡在险地之内的一缕缕金光，如无意外，便是这位强者当年遗留于此的力量。
<br/>　　不过，这些力量，历经千万年而不散。
<br/>　　这位强者，在上古之时，到底是强到了何等地步。
<br/>　　萧逸再次摸了摸这些金光。
<br/>　　手掌，再次一穿而过。
<br/>　　毫无疑问，这些金光之内，蕴含庞大力量。
<br/>　　不过，武者却触之不及，也无法利用。
<br/>　　倒是这里的妖兽，生活在此无数年，渐渐习惯了这里的气息以及日夜交替之变化。
<br/>　　故身体能承受与吸收这些金光。
<br/>　　这也便是刚才那头伏息树蟒能吸收这些金光，并实力大增的原因。
<br/>　　不过，这些金光并不会一直存在。
<br/>　　待得夜色消散，笼罩整片险地的金光悉数消散，妖兽吸收的这些金光，也会一并散去。
<br/>　　另外，人类武者虽无法吸收这些金光，却不代表无法利用。
<br/>　　金光险地这里的妖兽，能吸收这里的金光。
<br/>　　故，将其斩杀后，及时取出内丹；此时，内丹中，已蕴含金光力量。
<br/>　　只需以炼药师的手法，立刻将内丹炼制成丹药，便能保留金光内的力量。
<br/>　　万金府内，便有专门的‘金光丹’售卖，价格不菲。
<br/>　　金光丹之功效，与妖兽吸收这些金光的效果一样，便是短时间内增加武者的实力。
<br/>　　这些传言，萧逸原以为只是传闻轶事。
<br/>　　但现在看来，应该都是真的。
<br/>　　这也是一支支猎妖队、风使队等等武者队伍，昼伏夜出，特地在晚上才倾巢而出，斩杀妖兽的原因。
<br/>　　毫无疑问，金光险地，是一险地，却也是一宝地。
<br/>　　……
<br/>　　萧逸手中拿着伏息树蟒的内丹，微微皱眉。
<br/>　　因为，此刻内丹上的金光，正以缓慢的速度，渐渐消散着。
<br/>　　如无意外，数分钟后就会消散殆尽。
<br/>　　这里的妖兽，能吸收这些金光。
<br/>　　但妖兽身死后，内丹上的金光，也会渐渐消散。
<br/>　　这也是为什么斩杀妖兽后，要尽快将其炼制成丹药的原因。
<br/>　　这是唯一能保留这些金光力量的方法。
<br/>　　“难怪金光险地这边如此多武者队伍前来历练。”萧逸自语一声。
<br/>　　嘭…
<br/>　　忽然，手中一道火焰凝聚。
<br/>　　一株株天材地宝，凭空而现。
<br/>　　天材地宝与内丹，一并扔入火焰中。
<br/>　　不多时，一颗浑圆丹药，被炼制而出。
<br/>　　不过是炼制一颗普通丹药罢了，以萧逸现今炼药本事，若非炼制的丹药品阶太高，或者数量太大，根本无需祭出炼药炉。
<br/>　　萧逸看了眼丹药。
<br/>　　单从外边看，这粒丹药，并无太大特别。
<br/>　　这只是粒普通的王品丹药。
<br/>　　萧逸所用之材料、手法，也只是炼制普通增幅修为丹药的办法。
<br/>　　正常来说，若是加入的是正常的圣王境内丹，炼制出来的，只会是一颗王品的增幅修为丹药。
<br/>　　而现今，这粒丹药上，虽表面如常，但散发着的气息中，明显带着一股属于金属性气息的锋锐。
<br/>　　这代表着，这已经是一粒王品级别的金光丹。
<br/>　　萧逸张嘴服下。
<br/>　　几乎是丹药入腹的瞬间，萧逸身上气息，霎时爆发。
<br/>　　“达到圣王境八重巅峰了。”萧逸感受了一番此时力量，面露惊色。
<br/>　　如此惊人的效果，甚至比一些顶尖秘法还要强。
<br/>　　不过，仅仅是十分钟后，萧逸的气息便恢复了正常。
<br/>　　“金光丹的效果，只能维持十分钟左右。”萧逸自语一声。
<br/>　　这应该与金光丹的品阶，以及炼制时加入的内丹层次，还有自身修为强弱等等因素有关。
<br/>　　不过毫无疑问，金光丹的效果，惊人无比。
<br/>　　据他所知，万金府内售卖的金光丹，一粒王品等级的，便要50块灵石。
<br/>　　若是皇品等级的，起码售价500块灵石，甚至于若效果更大，其售价达到千块灵石以上。
<br/>　　这是相当昂贵的价格。
<br/>　　但换一方面想，这金光丹，绝对是不可多得的保命之物。
<br/>　　武者在外行走，无论是打斗还是逃命，一粒金光丹，绝对足以扭转战况。
<br/>　　这样算来，金光丹倒是值这个价。
<br/>　　“一粒王品的金光丹，便可卖50灵石。”
<br/>　　“一粒皇品级别，最差都能卖500灵石。”
<br/>　　萧逸自语一声，忽然眼前一亮。
<br/>　　这些妖兽内丹，斩杀妖兽后，内丹上的金光，很快就会消散。
<br/>　　故，来此的武者队伍，几乎都会带上一位炼药师。
<br/>　　斩杀妖兽，取内丹，随后立刻炼制成丹药。
<br/>　　可他萧逸，独自就能完成这些步骤。
<br/>　　以他实力以及炼药本事，怕是在此历练一月，便收获巨大。
<br/>　　斩杀的妖兽，妖兽精血可直接吸收，以修炼修罗战体。
<br/>　　得到的内丹，则可立刻炼成金光丹，之后卖出去，再去买大量妖兽精血。
<br/>　　萧逸的脸色，愈发欣喜。
<br/>　　“看来，修罗战体很快便能突破第九层，踏入第十层了。”萧逸笑笑。
<br/>　　嗖…身影一闪，瞬间消失在原地。
<br/>　　萧逸开始深入这片金光险地，一路前行，亦一路猎妖。
<br/>　　数个时辰后，萧逸渐渐深入。
<br/>　　所斩杀的圣王境妖兽，已不下数十。
<br/>　　又是半个时辰后。
<br/>　　萧逸的身影，忽然停顿。
<br/>　　前方，一头庞然大物，正散发着凶光，紧紧地凝视着他。
<br/>　　“金光幽狼，皇境妖兽。”萧逸双眼一眯，手中冷焱剑凭空而现。
