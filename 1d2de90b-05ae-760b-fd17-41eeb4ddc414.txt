    下山后，四人又在周围转了一圈，发现了一些车痕。<br/>    <br/>    阿乔木端详了一会儿，判断道，“是武装蟋蟀。”<br/>    <br/>    “好家伙，居然是武装蟋蟀，该不会是个揽私活的吧！”卡西惊叹道。<br/>    <br/>    陈兴走过去看了看，轮胎的纹理呈鱼鳞形状，确实是武装蟋蟀特有的。<br/>    <br/>    武装蟋蟀是红土重工生产的中高级全地形越野车，素有“荒野跑车”之称，用的不是传统的充气内胎或是真空胎，而是填充着鸽蛋状的“暗域橡胶纤维气泡”的超级胎，既能满足传统轮胎的弹性需求，又特别耐用。但由于内部力学结构的问题，会产生鱼鳞状的胎痕，非常具有标志性。<br/>    <br/>    武装蟋蟀价值不菲，能装备这样的车辆，代表佣兵团的实力不低。卡西所说的“揽私活”，指的是佣兵团里的人在没有经过上层的同意下，私自出来干活。能开武装蟋蟀的，十有八九和团长有密切的关系，陈兴立即做出了判断：<br/>    <br/>    “撤！”<br/>    <br/>    他们必须尽快离开，否则佣兵团回来寻仇，可不是他们几个人能挡得住的。在场的人都是老鸟，话不用多说，心里都明白。迅速返归营地，让商队连夜赶路。<br/>    <br/>    可到第二天下午，还是被佣兵团追上了。卡车的行进速度远不及越野车，车身又重，胎痕清晰无比。<br/>    <br/>    “黑鹫团办事，前方车辆马上停车，否则格杀勿论！黑鹫团办事，前方车辆马上停车，否则……”<br/>    <br/>    无线电短波公用通讯频道里传出警告的声音。转头看去，远处沙尘滚滚，似乎有大批车辆追来。陈兴组装好狙击枪，正准备趴下瞄准，却在这时，对讲机里传出胖子头领急促的声音：“停车，不要开枪，不要反抗！”<br/>    <br/>    商团里一共就五个佣兵，之前死了一个，现在只剩下四个，根本没有胜算。胖子头领选择了投降，按照行规，只要商队投降，除了钱财和货物被洗劫外，人可以活下来。<br/>    <br/>    货物来说，都是些边缘镇收回来的破烂，准备送去废品回收厂的，不值几个钱。现金他也没有多少，至于账户上的钱，没有黑表的流民无法过账，有黑表的公民，哪敢轻易留下过账记录？<br/>    <br/>    看着缓缓停下的车队，陈兴皱着眉头，稍微犹豫了几秒。背起狙击枪，跳下卡车，将胖子头领从商队里唯一一辆吉普车上拉下来，翻身上车。<br/>    <br/>    胖子头领有些理亏，也没说什么，任由陈兴上车。<br/>    <br/>    “大哥，带上我！”卡西看见，三两步跑过来，窜上副驾驶位。<br/>    <br/>    陈兴转头看向不远处的阿乔木和白人佣兵，两人都显得有些迟疑，似乎不想单独离开。<br/>    <br/>    就这么几秒钟里，黑鹫团的车辆出现在视野中，陈兴等不及了，油门一踩，调转车头，朝侧面冲去。先不说他身上带有价值昂贵的收藏品，那两个枪手都是他杀的，别人投降或许能保命，他肯定不行。而更重要的是，他没有把命运交到别人手里的习惯。要他坐以待毙，不可能！<br/>    <br/>    开出几百米后，黑鹫团的几辆沙地车追了过来，并伴随着尖锐的破空之声。<br/>    <br/>    “咻咻咻……”<br/>    <br/>    子弹从旁边擦过，陈兴看了卡西一眼，反身扑进后座。卡西立即翻进驾驶位，接手方向盘。<br/>    <br/>    陈兴迅速灌入灵能，弹匣内的子弹躁动起来。这把M4经过阿丽雅的改造，附带有火焰效果，对付机械一流。<br/>    <br/>    “哒哒哒！”<br/>    <br/>    一梭橙红的子弹扫出去，追在最前头的沙地车“轰”的一声，火焰冲了起来。接着车头一歪，翻滚起来。<br/>    <br/>    “哇噢！”卡西怪叫了起来，“Mr秦，你真的太厉害了！”他用力地拍打着方向盘，兴奋无比，“我爱死你了！”<br/>    <br/>    “闭嘴……”陈兴沉声说道，“如果你不想屁股开花的话。”<br/>    <br/>    “Mr秦，我是你唯一的队友，唯一的！”卡西大叫道。<br/>    <br/>    两人说话的时候，又一辆沙地车着火燃烧，惨叫声夹杂在呼啸的风中。<br/>    <br/>    紧接着，陈兴感到头皮一痛，向下躲闪。<br/>    <br/>    “铛铛铛！”<br/>    <br/>    脑袋刚缩回去，车尾就火花四溅，破空之声在头顶擦过。与此同时，卡西后脑勺蓝光晃动，一头撞在方向盘上。<br/>    <br/>    “我中弹了，我中弹了！”<br/>    <br/>    卡西慌乱地叫
