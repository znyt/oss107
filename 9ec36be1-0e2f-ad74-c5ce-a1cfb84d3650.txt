<br/>    风沙卷着白骨从天空中落下，白骨撞在沙面上惊起一片碎骨。
<br/>    
<br/>        狂风席卷着黄沙在天空之中形成一张巨大的脸，遮蔽整个天空，黑暗将一切笼罩，风妖老祖的声音从黑暗之中传来，“现在，人类，说出你的条件吧！”
<br/>    
<br/>        陈明的手中出现一团烈焰，照亮自己的身旁十余丈，掌中的火焰在跳动，“或许，我们可以商量一下。”
<br/>    
<br/>        直接要对方臣服，显然不可能，陈明道：“我想要的不多，我要整个御剑宗，以及，你不能伤害我燕山的人。”
<br/>    
<br/>        风妖老祖扫了陈明一眼，道：“你知道的并不够多，而且，你的价钱很合理，这是我能够接受的价钱。”
<br/>    
<br/>        陈明伸手，直接将手中的定风珠丢了出去。
<br/>    
<br/>        定风珠落入黄沙之中，瞬间消失不见，“还真是熟悉的味道啊，我，黄沙与狂风的君主，风扬，又回来了！”
<br/>    
<br/>        从狂风之中走出一个看起来如同黄沙构造而成的人形，“我曾经答应过御剑宗，绝对不会超过他们划线处一步，我便此生都未离开，既然已经和你达成了交易，那我也不会反悔！”
<br/>    
<br/>        黄沙在陈明的身前散落，一堆黄沙出现在陈明的身前，“这是我的残躯，只要将这些黄沙带上一颗在身上，我的孩子们都会束手，黄沙所及之处，风自然会停。”
<br/>    
<br/>        陈明伸手，将面前一堆黄沙全部收藏在储物戒中，然后他拿出四颗，给了卓青瑶四人，自己带了一颗，“走吧，去看看那御剑宗。”
<br/>    
<br/>        五人朝着风沙之中走去，果然，狂风遇到他们之后，自然退去，没有任何阻碍，在陈明的面前，黄沙形成一副地图。其中描绘了御剑宗，御剑宗就在这沙海的中央，而真正的沙海，足足有着十万八千里方圆，十万大山之中的沙海，不过只是一角而已。
<br/>    
<br/>        陈明带着四个亲传弟子朝着御剑宗的方向飞去，御剑宗深处沙海之中，门下自然也是有弱小的弟子，像是弱小的弟子，可能一辈子都无法离开沙海，所以御剑宗便设立了一个个风洞，让这些弟子可以尽快在沙海之中穿梭。借助御剑宗曾经留下的风洞，几人乘着狂风前行，三天之后，终于是来到了御剑宗之前。
<br/>    
<br/>        不过眼中所及，不过只是一片黄沙，似乎是感受到了陈明等人的到来，黄沙逐渐起卷，朝着周围散开，一个被埋葬在黄沙之中的古宗，逐渐出现在陈明的眼中。
<br/>    
<br/>        黄沙如同水流一般倾泻而下，一座巍峨的山门出现在陈明的眼中，那是一柄足有百丈高的青铜巨剑，横亘在天地之间，在巨剑之下，一块山石之上，刻着三个字，御剑宗。
<br/>    
<br/>        黄沙在陈明的身旁凝聚，“看到那一柄巨剑了吗？”
<br/>    
<br/>        陈明点头。
<br/>    
<br/>        黄沙之中传来风妖老祖的声音，“那就是神兵榜上第三位，巨阙。”
<br/>    
<br/>        陈明问道：“古仙门，又或是其他强大的宗门，应该知道巨阙在这里吧？为什么没有人来带走它？”
<br/>    
<br/>        黄沙之中飘过一道声音道：“因为他们都不可能得到巨阙的认可。”
<br/>    
<br/>        陈明问道：“想要让巨阙认可很难吗？”
<br/>    
<br/>        风妖老祖道：“哈哈哈哈，其实也并不是很难，只要能够拿得动这柄剑便可以了，所谓巨阙，便是硕大的山，这巨阙高一百零八丈，一丈便是一座山岳，巨阙炼制之时，取的是这大陆之上最为高大的一百零八座山岳，此后，这一座大陆再无一山可称山岳。就是我，也不能撼动这柄剑一丝一毫，当年，若不是因为这柄剑，我可不会败在御剑宗那个老家伙的手里。”
<br/>    
<br/>        卓青瑶看向巨阙，两眼之中闪烁着莫名的光彩，只要力气足够大就行了吗？
<br/>    
<br/>        陈明看向这巨阙，顶端已经深入云层之中，走到巨阙之前，陈明伸手抚摸着巨阙，巨阙忽然有了微微的颤动。
<br/>    
<br/>        风妖老祖的目光颤了颤，“这不可能！”
<br/>    
<br/>        陈明微微点头，看来要让巨阙认可也不是那么难，陈明飞身而上，直入云端，伸手握向那剑柄，体内的气血开始彭拜，血液在经脉中流动的声音如同沧海排浪，如同雷鸣般闷响。
<br/>    
<br/>        “给我起！”
<br/>    
<br/>        巨阙逐渐升起，带动整个大地微颤，轰隆隆！
<br/>    
<br/>        巨阙升高，彻底脱离了地面，陈明想要挥动手中百丈长的巨剑，却是太难，这也太重了，实际上，更多是因为巨阙认可了自己修行的九九玄功的缘故。
<br/>    
<br/>        不愧是只修行闻道境，只修肉身，修行肉身极致，追求肉身成圣的功法！
<br/>    
<br/>        巨阙在陈明的手中缩小，化作一柄比陈明还要高一个头的重剑，陈明伸手握住剑柄，从天空之中落下，直接伸手丢向卓青瑶，“拿去试试，看看趁手不趁手？”
<br/>    
<br/>        风妖老祖嗬嗬怪笑，如同风啸之音，“你能够得到巨阙的认可，已经是运道了，这个小姑娘也能得到巨阙的认可不成？”
<br/>    
<br/>        卓青瑶伸手，体内气血滚滚而动，如同雷鸣般炸响，如同洪钟震耳，其他三位亲传弟子，在卓青瑶的气血之下后退了几步，“大师姐，还真是一如既往的恐怖。”
<br/>    
<br/>        巨阙落入卓青瑶的手中，并没有半分不安的情况，在卓青瑶的手中，明显比在陈明的手中更加温顺。
<br/>    
<br/>        卓青瑶拿起巨阙，变小的巨阙，重量明显没有之前那么恐怖，在手中挥动了一番，而后将其背在了背后，朝着陈明拱手道：“多谢师父！”
<br/>    
<br/>        风妖老祖愣了，这么多年，没有一人能够拿走巨阙，今日倒是奇怪，师徒两个都能拿得动。神兵认可，就像是巨阙这般，并不是说一定要能够撼动一百零八座山岳才能拿动，它是按照境界来的，比如说卓青瑶现在的修为，换算成正常的修行境界，就是入道境巅峰，那么巨阙的要求就是正常的入道境巅峰的要求。
<br/>    
<br/>        而卓青瑶，同境界之中，整个大陆，恐怕都找不出第二个肉身有她这么强的，毕竟其他人都只有十条经脉，卓青瑶是八十一条经脉，自然是轻松达到了这个要求。
<br/>    
<br/>        而陈明，他的境界，自己都不明白，但是自己并不能挥动巨阙，可能是因为自己九九玄功的修为对于自己现在的实力来说还是太低了。
<br/>    
<br/>        有了这一柄巨阙，现在的卓青瑶，想想都觉得恐怖。
<br/>    
<br/>        陈明修行的九九玄功，和卓青瑶比还是差了点东西。怎么说呢，陈明虽然也能够修行，但是毕竟体质不是完美的符合九九玄功，并不能发挥出九九玄功最大的威力。
