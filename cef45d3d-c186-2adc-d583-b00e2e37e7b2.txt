　　“我……”
<br/>　　庄菲菲顿时也不知道该说什么了，只好望向一旁的叶尘。
<br/>　　事到如今，她也只能完全交给叶尘处理了。
<br/>　　不料，
<br/>　　叶尘缓缓站起身来，嘴角微微一扬，冷冷道：
<br/>　　“和谈？你倒是想的挺美！”
<br/>　　“你先是利用职务之便，想要对庄菲菲潜规则，遭到拒绝之后，便设下陷阱，害的他险些在非洲丧命，如今居然还腆着脸，要她赔偿违约金，你还真是够无耻啊！”
<br/>　　庄菲菲的事情，叶尘之前已经听她说过，若非看她被逼的实在可怜，叶尘才懒得亲自处理这种鸡毛蒜皮的小事。
<br/>　　高总监听到这话，脸色顿时一阵变幻，继而有些恼羞成怒，
<br/>　　“好小子！你竟然敢污蔑本总监？菲菲小姐，我不管这小子是你什么人，如果你还想继续在娱乐圈混下去的话，就立刻让这小子给我滚蛋！否则你可休怪我不讲往日的情面！”
<br/>　　旁边的那名美女律师，此时也急的不得了，当即扯了扯庄菲菲的衣袖，低声道：
<br/>　　“菲菲姐，你还是赶紧向高总监服个软吧！否则他真要是按照合约，把你告上法庭，咱们十有八九是要败诉的，那可是足足十倍的违约金，将近一个亿呢！”
<br/>　　庄菲菲听到自家律师这话，也不由得娇躯微微一颤。
<br/>　　她如今虽然是华夏娱乐圈的一线明星，可是毕竟根基尚浅，真要是和华艺天皇娱乐闹翻了，今后的星路必定会大受影响，而且她如今的全部身家，也不过就一两个亿而已，真要是一口气赔出去，那她就真的破产了！
<br/>　　纵然叶尘的身份非同寻常，可是娱乐圈的事情，他也未必能插得上手吧？
<br/>　　一念及此，庄菲菲不由得再次朝叶尘望了过去，
<br/>　　叶尘自然看出了庄菲菲的担忧，不由得微微一笑，
<br/>　　“庄小姐，你放心，此事我既然答应帮你解决，就一定会帮你处理的妥妥当当！”
<br/>　　说到这里，叶尘指了指高总监，淡淡道：
<br/>　　“这个什么狗屁总监，不过就是一个小喽啰而已，和他也没什么好谈的，咱们还是等杨国强来了再说吧！”
<br/>　　高总监听到前半句，顿时怒发冲冠，气得连杀人的心都有了，不过听到半句，尤其是听‘杨国强’三个字，整个人又顿时一滞，眼眸中露出一抹惊疑之色，忍不住道：
<br/>　　“你刚刚说……杨、杨董事长要来？”
<br/>　　叶尘连理都没理，继续低头品茶。
<br/>　　高总监顿时有些懵了，一时间竟也看不出叶尘的深浅，更加不敢轻举妄动。
<br/>　　娱乐圈鱼龙混杂，想要在这个圈子里混下去，就要有一双火眼金睛，知道什么人能得罪，什么人不能得罪。
<br/>　　高总监能混到华艺天皇娱乐的高层，自然也是有些真功夫的。
<br/>　　虽然叶尘看上去年纪轻轻，可是气度颇为不凡，真要是某位超级大家族的公子哥，想要为庄菲菲出头，他还真得罪不起。
<br/>　　只不过，能请动他们华艺天皇娱乐的幕后大老板，那此人的背景得有多恐怖？
<br/>　　庄菲菲何时攀上了这么一位大人物？他怎么从来都没有听说过？
<br/>　　一念及此，高总监当即悄悄退到一旁，取出手机，拨通了一个号码，脸色顿时洋溢着谄媚的笑容，
<br/>　　“喂，张哥吗，我小高啊！向您打听个事，我听说，今天杨董事长，要来钱潮路红玫瑰餐厅，我刚好也在这儿，不知道杨董事长在哪个房间，我想过去敬杯酒……”
<br/>　　原来，
<br/>　　这位高总监口中的‘张哥’，是杨氏集团总部的一位高层。
<br/>　　高总监当初下了大功夫，这才结交上这位贵人，算是他的靠山，也正因为如此，他才能在华艺天皇娱乐平步青云，坐到总监的位置。
<br/>　　高总监旁敲侧击的询问了一番，电话那头顿时传来一道不耐的声音，
<br/>　　“小高，你这是从哪听来的消息？今天杨董事长要和米国埃米尔集团总裁库里先生会晤，这可是我们杨氏集团本年度最重要的大事！怎么可能会去什么钱潮路？真是不知所谓！”
<br/>　　高总监被电话那头的‘张哥’一通训斥，非得一点不生气，反而心头大喜，挂了电话之后，望向叶尘的目光中，露着一抹浓浓的嘲讽之色，冷冷道：
<br/>　　“好小子啊！若非老子在总部有靠山，还真被你给唬住了！”
<br/>　　“杨董事长日理万机，是何等的身份？会来见你一个乳臭未干的毛头小子？”
<br/>　　庄菲菲三人，顿时全都脸色大变，
<br/>　　刚刚高总监和那位集团高层的通话，因为声音很大，所以她们都听的仔细。
<br/>　　就连庄菲菲，原本还对叶尘充满期望，此时也不由得有些动摇了，
<br/>　　纵然叶尘的身份再不凡，那杨国强应该也不至于会放下如此重要的客户，跑来管这等小事吧？
<br/>　　那位美女律师，望向叶尘的目光中，更是充满着浓浓的怨恨，
<br/>　　若非叶尘信口开河，把局面搞得如今这个样子，说不定她们已经和高总监达成和解了。
<br/>　　如今骑虎难下，又该如何收场才好？
<br/>　　三人的神情，落在高总监的眼中，他顿时更加确信了自己的判断，不由得嘿嘿一笑，
<br/>　　“菲菲小姐，本总监就再给你一次机会，只要你把这个，不知道天高地厚的臭小子，给我赶出去！而且今后凡事都乖乖听从我的安排，违约金的事情，我可以做主，不再追究，而且保证不会亏待你，如何？”
<br/>　　庄菲菲自然听出了高总监话里的意思，俏脸顿时微微发白。
<br/>　　她今日若是答应了，以后怕是就要彻底沦为高总监的玩物了！
<br/>　　叶尘听到这话，不由得两眼微微一寒，已经动了一丝杀意。
<br/>　　就在他懒得和这个高总监废话，准备直接一道神念将其灭杀的时候，门外忽然传来了一道躁动之声。
<br/>　　继而，
<br/>　　开始有人敲门，
<br/>　　同时还有一道浑厚的声音响起，透着恭敬，
<br/>　　“请问，叶仙人在里面吗？杨氏集团杨国强特来拜见！”
<br/>　　随着这道突兀的声音响起，整个包厢里的所有人，顿时全部惊呆，
<br/>　　“杨氏集团董事长，杨国强？他不是在会见一个重要的客户吗？怎么会突然来此？还有，他口中的叶仙人，又是何人？”
