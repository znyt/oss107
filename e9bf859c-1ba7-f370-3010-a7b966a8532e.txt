　　一直到第二天的晚上，刘哔都守在木棺旁边。
<br/>　　白天的时候文才和九叔出去了一趟，说是去了任家，因为担心义庄发生什么意外，刘哔并没有一起跟去。
<br/>　　但不知道为什么，从任家回来之后，九叔看自己的眼神就有些不太一样了。
<br/>　　虽然来到这个世界的时间不长，但九叔从来没有像现在这样，看着他的眼神，如此戒备和陌生。
<br/>　　终于还是，发现了什么吗？
<br/>　　心中叹了口气，刘哔也下定了决心，等这里的事情全部解决之后，他就找个机会离开这里。
<br/>　　一直到现在，他都没能看透九叔的实力，但他无比确定的是，如果九叔和他交手，他是不可能有一点胜算的。
<br/>　　他也不敢保证，九叔一定不会对他动手，人心隔肚皮，如果真发现了他的身份，谁知道九叔会做出什么样的选择呢？
<br/>　　关乎性命的事情，他可不敢冒险赌博。
<br/>　　等到深夜，万籁寂静的时候，刘哔盘坐在一张木椅上，手边放着一柄看起来有很多年份的桃木剑。
<br/>　　文才和九叔早就已经熟睡，整个义庄都没有一点声音，就连平日里那些令人烦躁的昆虫叫声，也意外的全部消失了。
<br/>　　此时刘哔手中掐着一个奇特的法决，正将附近的天地灵气缓缓聚集在自己的丹田了，一天的修炼，让体内的深红色灵种又增大了一些。
<br/>　　突然一道紫光从刘哔胸口闪过，感受到胸前有些发烫，他睁开了眼睛，将画像从衣服内掏了出来。
<br/>　　“有事？”看着画卷上的绝美女子，刘哔皱了一下眉头，修炼状态被打断，让他心中有些烦躁。
<br/>　　“你旁边有一只僵尸。”听到刘哔有些不快的语气，紫女有些疑惑，不知道她又怎么得罪这家伙了。
<br/>　　“我知道。”刘哔点了点头，脸上并没有太大的波动。
<br/>　　看到刘哔如此淡定，紫女却是有些坐不住了，语气急切的开口道：“知道你不去解决了它？你到底有没有将破除封印的事放在心上！”
<br/>　　有些懵逼的看了紫女一眼，他心中有些奇怪。
<br/>　　“这家伙才死了二十多年，帮你解除封印不是需要百年尸丹吗？”看着紫女，刘哔继续说道：“不符合条件，你着什么急。”
<br/>　　撇了撇嘴，紫女似乎想到了什么，耐心的和刘哔解释道：“按照常理来说，是这样，但是这个结果不同。”
<br/>　　停顿了一下，紫女将原因说明：“这个僵尸体内被人种下了鬼气，和本身的尸气融合，正好形成了一枚伪百年尸丹，虽然效果并没有真正的百年丹那样好，但破除封印却已经够了。”
<br/>　　听完紫女的解释，刘哔心中也有些惊讶，一时之间也没有回话，脑海中像是在思考着什么东西。
<br/>　　“喂喂喂！你到底有没有听我说啊！你这家伙，快把这头僵尸解决掉啊！”紫女看刘哔不做反应，呆愣愣的样子，有些气结。
<br/>　　真是的，她怎么找了这么一个合作伙伴，以前怎么没发现，这家伙反应慢的要命。
<br/>　　“别吵！”脑海中的思路被紫女打断，刘哔有些不太开心，低喝一句，双眼依旧盯着木棺，两条清秀的眉毛紧紧蹙在一起。
<br/>　　看到刘哔脸上认真的表情，紫女知道自己可能打扰了他，吐了吐舌头，乖巧的应了一声。
<br/>　　没有理会紫女，刘哔心中梳理着仅有的几条线索，一个不太成型的猜想在他脑海中缓缓呈现着。
<br/>　　木棺中有人家的东西，这一点他无比确定，但是具体是什么东西，他之前却一点头绪都没有。
<br/>　　但结合紫女刚才所说，他却有了线索，任家藏在尸体里的东西，是否就是这颗鬼气和尸气结合在一起打造出来的尸丹？
<br/>　　仔细斟酌，他觉的自己的猜想八九不离十，但这颗尸丹，是要送去什么地方呢？
<br/>　　肯定不会是任家自己所需，信上的内容虽然不多，但已经点出有一件货物要运向不知道什么地方。
<br/>　　假设说尸丹就是货物，那么收货人又是谁呢？
<br/>　　寻常人肯定是用不到这种东西的，只有修炼鬼术的道士，才可能在某些地方用到，除此之外，也只有鬼和僵尸有用了。
<br/>　　而任老爷子体内多出来的这股鬼气，这并不正常，僵尸体内不可能出现鬼气，除非是是有人故意灌注进去。
<br/>　　难道说，和任家做交易的，是一个鬼修，或者直接是某个鬼怪？
<br/>　　这个人给任家提供鬼气，任家帮他找一些已经死去的尸体，然后用鬼气强行催化出伪尸丹，给幕后那人提供修炼资源？
<br/>　　顺着脑海中的思路，刘哔大胆的做了一个假设，如果是这样解释的话，发生这种事情也就不怎么离奇了。
<br/>　　但原著中，任发没想到的是九叔这边居然会出现了纰漏，将这头百年僵尸放了出来，最后惨死。
<br/>　　但是这件事情还是有很多漏洞，事实的真实性，刘哔也不敢贸然下什么决定，他也不准备自己掺和进去。
<br/>　　毫不客气的说，这件事和他没有半点关系，就算任家真的和鬼修做交易，并且倒卖私人尸体，那也不应该他去管。
<br/>　　之所以现在还守在这里，也是为了系统给出的连环任务，否则他才不会这么上心。
<br/>　　探查任家，也是为了搞明白任家的底细，为接下来的任务做一些准备而已。
<br/>　　如果接下来的任务和这件事情有关联，那么有手上掌握的这些线索，他也不至于闭眼抓瞎，如果和这件事无关，那就最好不过了。
<br/>　　对于那些手段诡异的鬼修，刘哔可不想和他们沾上什么关系，不管是友是敌，都会给他带来连绵不断的麻烦。
<br/>　　“砰砰！”
<br/>　　就在刘哔低头冥想的时候，木棺突然晃动了一下，表面上缠绕的墨斗，通体散发着金色的光辉，像网一样将棺材包裹着。
<br/>　　擦了擦手，他拿起一旁古旧的桃木剑，将体内的灵气不断向上面灌输着。
<br/>　　似乎是受到刘哔灵气的影响，桃木将通体都染上了一层血一般的猩红色，紫女在一旁看着，眼中露出了一抹惊疑。
