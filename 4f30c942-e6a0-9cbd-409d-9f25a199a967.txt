　　很快，光明教的人马，也已经呼啸至跟前，
<br/>　　为首的一人，身穿金衣长袍，宛如中世纪的君王，而且身上的气息，丝毫不弱于魔祖和九首妖皇。
<br/>　　“神王！”
<br/>　　叶尘一眼便将此人认出，赫然也是昔日在华山之巅，被自己诛杀的神界第一强者神王。
<br/>　　“我的天呐！三大化神境强者，上百名元婴强者！”
<br/>　　“这等阵容，即便是将整个地球毁灭，也是轻而易举的事情吧？”
<br/>　　“完了！完了！没想到三大势力，竟然全部出动，这还怎么打？”

<br/>　　东海沿岸，华夏众武者，个个脸色土灰，一些胆子小的，已经开始悄悄往后方倒退，甚至直接远遁而去。
<br/>　　叶尘从始至终，一脸淡漠，表情无悲无喜，似乎在思索着什么。
<br/>　　这落在众人的眼中，显然以为叶尘已经认命，不由得再次纷纷叹息。
<br/>　　而直播间里的亿万观众，也全都一阵绝望，
<br/>　　“本以为叶仙人就算不敌，至少也能支撑片刻，如今看来，怕是没戏了！”
<br/>　　“我早就说过，叶狂仙此举，简直就是自取灭亡啊！”
<br/>　　“可怜我华夏，也要跟着他一起遭殃了！”
<br/>　　……
<br/>　　网络上的舆论，几乎是一边倒，之前还有抱有希望的那些人，此刻也全都纷纷叹息抱怨起来。
<br/>　　整个华夏此刻，全都陷入了无尽的压抑情绪之中。
<br/>　　“传我的命令下去，华夏沿海防线，进入一级戒备状态！”
<br/>　　就连易山河，也已经悄悄下达了一道军令，以防止叶尘落败之后，三大势力入侵。
<br/>　　“小白，你说叶尘他，能赢吗？”
<br/>　　就连一直对叶尘自信满满的唐清雅，看到眼前的这一幕，也不由得秀拳紧握，娇躯微微颤抖，美眸中透着浓浓的担忧之色。
<br/>　　并非他不相信叶尘，实在是对方的实力太强了！
<br/>　　那可是三名化神强者啊！
<br/>　　普通人或许不清楚，可是她身为修真者，又岂会不知道，化神之下，皆为蝼蚁！
<br/>　　化神境和元婴巅峰，虽然只有一步之差，实力上的差距，却天差地远，甚至一名化神境，可以吊打十名以上元婴巅峰！
<br/>　　更何况，眼下的这阵势，除了三位化神强者之外，还有足足上百名的元婴强者！
<br/>　　祝小白同样脸色铁青，深吸了一口气，方才缓缓道：
<br/>　　“清雅姐，这个……我也说不好……”
<br/>　　祝小白自己本身，就已经一只脚迈入元婴境，自然知道三名化神强者，和一百名元婴强者，这个阵容是何等恐怖！
<br/>　　纵然他一向对叶尘有着绝对的信心，此刻心里也不免有些打鼓了。
<br/>　　“桀桀！叶狂仙，你怎么也没有想到，我们三方势力会倾巢而出吧？”
<br/>　　九首妖皇北面空中而立，高高俯视着下方的叶尘，满脸的讥讽之色。
<br/>　　“十二年前，你灭我三界之仇，今日这笔账，也该好好算一算了！”
<br/>　　九首妖皇的声音刚刚落下，南面的神王，也跟着缓缓开口道。
<br/>　　“当然了！如果你愿意跪下磕头求饶，我们或许可以考虑饶你一命！”
<br/>　　说话的是魔祖，眼眸中透着浓浓的嘲讽、戏谑之色。
<br/>　　九首妖皇和神王，也纷纷抚掌大笑，
<br/>　　“不错！只要你跪下磕头求饶，我们往日的仇怨，可以就此揭过！”
<br/>　　东海沿岸的华夏众武者，以及直播间的亿万观众，听到三大化神强者这话，顿时一个个的眼眸中，又重新燃起了希望，纷纷低呼起来，
<br/>　　“跪吧！”
<br/>　　“向三位化神强者磕头求饶，不丢人！”
<br/>　　“识时务者为俊杰，退一步海阔天空啊！”
<br/>　　“叶仙人，为了您自己，也为了整个华夏，您就忍辱负重一次吧！”
<br/>　　……
<br/>　　在众人看来，面对如此逆天的阵容，叶尘出来磕头求饶之外，恐怕再无任何活路。
<br/>　　然而，在众人充满期盼的目光中，叶尘终于缓缓抬起头，在三大化神强者身上，一一扫视了一遍，方才缓缓开口，
<br/>　　“就凭你们三个废物，也想让我下跪？你们是说梦话吗？”
<br/>　　轰！
<br/>　　叶尘这话声音不大，但是他这一出口，顿时一道春雷，在众人的脑海中爆炸开来，一个个全都懵了！
<br/>　　“他竟然敢说三大化神强者是废物？”
<br/>　　“他是疯了吗？”
<br/>　　“狂妄！”
<br/>　　“找死！”
<br/>　　……
<br/>　　不光围观的华夏武者们，一个个全都脸色狂变，甚至都怀疑自己的耳朵出了问题。
<br/>　　三大势力一方的强者们，反应过来之后，顿时一个个气得肺都要炸了，尤其是魔祖、神王和九首妖皇三人，身上的气势，顿时便攀升到了巅峰。

<br/>　　“这个叶狂仙，只会逞口舌之力，却将整个华夏都拉入了万劫不复的境地！”
<br/>　　“疯子！他就是一个疯子！”

<br/>　　直播间里此刻也炸开了锅，几乎全都在抨击叶尘此举，简直就是在自寻死路。

<br/>　　恐怖至极的一刀，朝下方的叶尘，狠狠劈落了下来！

<br/>　　魔祖的这一击，震撼住了所有人，就连之前还心存一丝侥幸心理的唐清雅和祝小白，也不由得脸色沉重起来。

<br/>　　只见，面对魔祖那仿佛能毁天灭地的一击，叶尘只是缓缓抬起一只手臂，随手一拍，
<br/>　　嘭嘭嘭！
<br/>　　那长达数千米的恐怖血刀，竟然犹如纸糊的一般，瞬间碎裂开来，消散于无形。
