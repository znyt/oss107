<br/>        陈明微微点头，“既然有这么一个地方，那就去看看吧。”
<br/>    
<br/>        星冥道：“道友，我劝你还是三思而后行，这一次前去争夺宝物的，可是有着不少的悟道榜上的大能前往，几乎都是有着号称诸王之首的存在。道友若是执意要去，最好还是让师门的悟道境长辈带着你们去。”
<br/>    
<br/>        陈明摆手道：“多谢老哥提醒了，不过，我自己心里有数，劳烦老哥问雪山老君要两个名额。”
<br/>    
<br/>        星冥疑惑道：“两个名额？”

<br/>        卓青瑶当即挺了挺小胸脯，看的陈明心头微震，这小妮子还真是越长越有女人味了，卓青瑶道：“我也要去！”

<br/>        星冥直接愣住了，“不是，道友，你可能有所不知，虽说你的徒弟，卓青瑶现在才是闻道境，年纪有些大，看起来天赋一般。但是历来能够在榜单上登顶的天才，都是一鸣惊人的，最后几乎都是一方大能，若是给卓青瑶时间，称王必定不在话下。加上有着巨阙，就是大君，也有着极大的可能性。但是卓青瑶现在，毕竟还是闻道境，你就把她带到诸王的争锋中去吗？”

<br/>        卓青瑶看向星冥，问道：“你这是在看不起我的实力？”
