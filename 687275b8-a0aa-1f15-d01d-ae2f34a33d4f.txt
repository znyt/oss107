　　第90章
<br/>　　千顷雪原泛夜光，诗情人意两茫茫，
<br/>　　前村五路凭君踏，路也迢迢夜也长。
<br/>　　白茫茫山川大地，四匹快马拉着木板改成的雪橇在雪地上飞驰，小小的黑点宛若宣纸上一滴黑墨。
<br/>　　程大雷裹着大氅坐在雪橇上，朔风扑面，眼睛眉毛都挂着白霜。他裹紧身上的大氅，口中悠悠念出四句诗。
<br/>　　张肥眼睛一亮：“大当家还会写诗！”
<br/>　　“哈，让你看出来了。”程大雷道：“我曾经也是读圣贤书，想着为国为民做一番事业，都怪这世道啊，生生把我这读书人逼成山贼。”
<br/>　　说这话时，程大雷脸不红心不跳。他看着张肥出神的模样，问：“莫非三爷也是同道中人？”
<br/>　　“这个……闲时却也好个文墨消遣。”
<br/>　　“啧！”
<br/>　　程大雷撇撇嘴，这还真是人不可貌相，谁能想到外表粗鄙的张三爷，内心还是个文艺青年。
<br/>　　马车拉着几人在雪地奔驰，速度快了一倍不止，今天天黑，就能到黑石城。
<br/>　　不过这天气可是真冷啊，窝在雪橇上手脚都快要冻僵了。程大雷心里忍不住想，自己不在山寨想清福，非要出来跑这一趟是不是脑袋被门挤了。
<br/>　　这次行动的目的，是去黑石城收割一波恐惧值。黑石城有十万人，只要一人给自己贡献一点也就够了。当然，帐不能这么算，每个人贡献的恐惧值也不止一点。
<br/>　　程大雷窝在雪橇上查看刚触发的任务。
<br/>　　嘟，抢劫城中富户10次。
<br/>　　嘟，正义的少年喔，身为山贼打家劫舍是必不可少的功课，如今城中有为富不仁的大户，主动出击去教训他们吧。
<br/>　　目前完成进度2|10 。
<br/>　　程大雷琢磨这2|10是如何来的，大闹黄家算一次，毕竟把柳芷抢来了，擒住杜茂算一次，自己把人都绑到山上了。
<br/>　　如此说，这任务也不难完成。
<br/>　　只是，每次系统出现这种中二语气，程大雷总有不详的预感。这次行动，该不会出什么岔子吧。
<br/>　　“秦蛮，咱们距离黑石城还有多远？”
<br/>　　“再有一个时辰准到。”秦蛮挥鞭抽在马上，道：“大当家，黑石城外有座土地庙，离城十里地，咱们可以在那里落脚。”
<br/>　　“好，到了先歇一天，天亮去城里摸摸情况。”
<br/>　　冬季天黑得早一些，众人到土地庙的手，天已经黑下来。寥廓的旷野上，孤零零一座土地庙，倒是一处藏身的好所在。
<br/>　　只是土地庙里有火光闪动，里面明显有人。
<br/>　　“大当家，咱们的天气被人占了。”秦蛮有些不快，随之道：“这样的天气，谁在这荒废的破庙中做什么？”
<br/>　　程大雷也皱起眉头，道：“秦蛮，军师，咱们三个进去，看看对方什么来路。”
<br/>　　几人在离庙五百米处停下，程大雷还是裹着大氅，以防万一他将自己的斧头也扛上了。
<br/>　　在三人走入土地庙的时候，庙内的火光突然熄灭，只是一刹那，夜幕就笼罩了三人。
<br/>　　对方反应很快，周围没有任何动静，隐隐约约似乎有长短不一的呼吸声。
<br/>　　程大雷握紧了斧柄，开口道：“那条道上的弟兄，咱们今天从这里路过，招子放亮些，别伤了绿林道的情分。”
<br/>　　沉默，瘆人的沉默。
<br/>　　过了半晌，黑暗中响起一个声音：“程当家……”
<br/>　　这声音听上去很熟悉，程大雷愣了愣道：“高二爷……”
<br/>　　咔！
<br/>　　一束火把被点燃，黑漆漆的庙里有了光亮，看清楚对面那人后，程大雷有些无语。
<br/>　　对方不是高飞豹又是那个。
<br/>　　高飞豹一行十几个人，身上都带着兵器。程大雷挥挥手，让林少羽三人过来。
<br/>　　庙里的篝火重新被点燃，渐渐有了暖意。双方围着火堆而坐，马也都被牵到庙里。
<br/>　　程大雷和高飞豹大眼瞪小眼，内心都有些崩溃。
<br/>　　“高二爷，我是真佩服你，这么冷的天也不歇着，您可真够敬业的。”
<br/>　　“哈，程当家不也一样嘛，天寒地冻也不在山寨想清福。”
<br/>　　“带兄弟出来活动活动筋骨，也不图发财。你们呢……”程大雷眨眨眼睛：“怎么，山寨没吃食了？”
<br/>　　“什么话！”高飞豹差点站起来：“飞虎寨什么时候穷过，我们也是在山寨太清闲了，出来活动活动。”
<br/>　　“二当家，外面又有人来！”飞虎寨在外面放风的人道。
<br/>　　“今天什么情况，赶集来得嘛！”高飞豹跳起来道：‘熄火！’
<br/>　　“等等。”程大雷隔着门口向外看了一眼：“先别着急，估计也是熟人。”
<br/>　　还真程大雷猜中了，新来的一股人在离庙门一百米处停下，似乎也看到了这里火光，不敢上前。
<br/>　　“白当家，进来吧，这里没外人。”
<br/>　　这次来的是杏花岭的小白狼，她也带了七八个手下。走进庙门的时候，看到程大雷和高飞豹两伙人，整个人也愣了好久。
<br/>　　“二位当家，还真是……敬业呐。”
<br/>　　庙里的人马分成三股，各自聚拢成一堆。程大雷靠墙坐着，心情很是无奈。
<br/>　　秦蛮递过来一个酒囊：“大当家，喝些酒暖暖身子吧。”
<br/>　　“让弟兄们也喝点，这天太冷了。”程大雷咕咚咚灌了一口，心里不仅有些怀念在山寨吃着火锅唱着歌的日子。
<br/>　　如今只能吃些冷面包，冻腊肉，天寒地冻，冷得咯牙，程大雷只好放在火堆边一点点烤着。
<br/>　　“程当家吃什么呢？”高飞豹突然问。
<br/>　　“还能吃什么，烤点面包，熏点腊肉，出门在外谁还不得受点罪。”
<br/>　　程大雷长长的叹口气，说罢耳边忽然响起吸溜口水的声音。他抬起头，发现小白狼和高飞豹两拨人都盯着自己，目光像盯着食物的狼。
<br/>　　“怎么，是我说错话了嘛？”程大雷挠挠头，忽然问：“你们吃什么呢？”
<br/>　　高飞豹没有回答，往高粱面里浇了些热水，用手握成面团，一点一点的往嘴里咽。
<br/>　　“高二爷，你们就吃这些！”程大雷难以置信道：“我懂了，这叫绿色有机，补充膳食纤维，对不对！”
<br/>　　“呵……呵呵。”高飞豹干笑两声，握着手中的面团，突然觉得有些难以下咽。
<br/>　　
<br/>(本章完)
