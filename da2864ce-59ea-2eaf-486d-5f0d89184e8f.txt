<br/>    金毛吼所想的，无非就是所有大妖王同仇敌忾，共同对抗妖族大君。
<br/>    
<br/>        金毛吼都：“传言毕竟只是传言，现在妖族大君还没有真正动手，我们便不知道其中的真假，但是，我们都要早作准备，若是妖族大君真的发难，我们都要出兵联合对抗妖族大君！”
<br/>    
<br/>        陈明微微点头，没有什么不能同意的，别说九个大妖王了，就是二十个大妖王，也不一定挡得住啊，一旦正面战场彻底崩溃，来的可就不仅仅只是一位大君了。
<br/>    
<br/>        大君到底有多强，陈明现在没有确切的概念，但是光是一句大君定国，就可以知晓一二。
<br/>    
<br/>        大君的战力，对于妖军来说，绝对是不可战胜的。
<br/>    
<br/>        在金毛吼这里签下了约定书之后，三位大妖王也是各自散去。
<br/>    
<br/>        回到自己的房间，场上正在拍卖一件不知名的宝物，那是一颗黄色的珠子，听拍卖师的介绍，这颗珠子，除了坚硬，似乎并没有什么特殊之处。
<br/>    
<br/>        然后陈明就看到李素衣叫价了，“三百二十万！”
<br/>    
<br/>        对面长河的房间之中，那个小王爷的声音传来，“三百三十万！”
<br/>    
<br/>        李素衣毫不迟疑，继续叫道：“三百四十万！”
<br/>    
<br/>        双方都在加价，陈明询问了一番这颗珠子的低价，凌仙道：“一百灵石。”
<br/>    
<br/>        陈明看了一眼，这情况不太对，一般这样便宜的东西，主角不都是轻轻松松就买下了吗？怎么还有人抬杠？
<br/>    
<br/>        陈明看了对面的房间一样，然后就听到那小王爷传出来痛苦的喊声，“父王，父王我错了，我再也不拍了！”
<br/>    
<br/>        好吧，原来是长河家那败家玩意儿，陈明看向李素衣，问道：“这颗珠子有什么用处？”
<br/>    
<br/>        李素衣道：“师父，你有所不知，这颗珠子，叫做定风珠，在沙海之中，有一种生灵，叫做风妖，而这珠子，便是风妖的内丹，而且是风妖老祖的内丹，只要我们将这颗珠子送到沙海之中，沙海之中风妖老祖，就会给我们一些好处。”
<br/>    
<br/>        嗯，这颗珠子居然有着这么大的用处！
<br/>    
<br/>        陈明问道：“那风妖老祖，到底是什么修为？”
<br/>    
<br/>        李素衣道：“这个弟子不知，不过那风妖得到了这定风珠之后，修为一旦恢复，恐怕有着大君的实力。”
<br/>    
<br/>        陈明估计了一番，估计那风妖老祖绝对不止大君的实力，否则，若是大君的实力有这般恐怖，那妖族大君早就动手了，还至于怕九个大妖王？
<br/>    
<br/>        如果真的有了这定风珠，倒是真的可以直接将整个燕山直接搬到那沙海的中心去，到时候，只要不是鬼仙降临，那么燕山可谓就是真的天地不动我不动。
<br/>    
<br/>        陈明微微点头，怪不得李素衣说什么也要拿下这颗珠子，“继续拍。”
<br/>    
<br/>        李素衣拱手道：“是，师父！”
<br/>    
<br/>        没有了小王爷的捣乱，李素衣倒是很轻松的拍下了这颗珠子。
<br/>    
<br/>        将珠子送到他们的房间，陈明给了灵石，将珠子接了过来，看向李素衣，问道：“你是想要那沙海之中的什么东西吗？”
<br/>    
<br/>        李素衣道：“师父，在那沙海之中，有着一个上古宗门，那试练塔，便是那宗门的一部分，而那风妖老祖，便是上古宗门的护山神兽，那一个上古宗门覆灭之后，风妖老祖也在那一战之中失去了自己的内丹，不复之前，若是我们燕山能够计入沙海，便可以得到那一处上古宗门的传承，对于门下的弟子来说，也是一个优点。”
<br/>    
<br/>        陈明微微点头，问道：“那个上古宗门之中，可有什么阵法？”
<br/>    
<br/>        李素衣摇摇头，道：“师父，这我就不知道了，里面究竟有什么，还要我们进去了才知道。弟子只是知道，这是一个关于剑道的宗门。”
<br/>    
<br/>        看来李素衣知道的也不多，不过这沙海之中，到底隐藏着什么秘密，还要等他们进去之后才知道。
<br/>    
<br/>        而且，李素衣是没有任何私心的，他自己并不清楚里面有什么，只是知道这是一个剑道宗门，估计是为了卓青瑶和陈灵鱼，还有自己着想，才想到拿下这颗珠子的。
<br/>    
<br/>        后面拍卖的东西，在陈明看来，都没有什么用处，倒是拍卖了几种阵法图，但是对于陈明来说，却是已经没有了任何用处。
<br/>    
<br/>        和几位大妖王私下谈妥了，定风珠也拿到了，陈明也是迫不及待，想要知道那个上古宗门之中到底留下了一些什么宝物，要是能够有一种阵法图，自己的实力可谓是又可以有一次大大的提升。
<br/>    
<br/>        带着血衣剑卫，和四位亲传弟子进入马车之中，他们准备回燕山，走了七八天，终于是来到了沙海之外，陈明让血衣剑卫在外等候，带着四个亲传弟子来到沙海附近，陈明也是从储物戒之中拿出了一颗珠子，便是那定风珠。
<br/>    
<br/>        陈明分明地看到，天地之间狂风暴卷，风沙袭来，不过最后却是停留在一根线之前，无论如何，风沙都没有越过那一根线。
<br/>    
<br/>        陈明站在线后，静静地等待，不过半刻钟，前方的天空之中，黄沙凝聚成一张巨大的脸，看起来就像是一张耗子的脸。“人类，你得到了不属于你的东西！”
<br/>    
<br/>        四位亲传弟子看到这般异象，都是微微有所震惊，不过毕竟是燕山的亲传弟子，也没有太过惊讶。
<br/>    
<br/>        陈明走到那一条线前，问道：“你便是风妖老祖？”
<br/>    
<br/>        风妖老祖哈哈大笑，黄沙从他的口中流淌而出，“人类，交出那一颗珠子，我留你一命！”
<br/>    
<br/>        陈明直接在这一条线之前坐了下来，道：“我既然拿出了这颗珠子，就知道这颗珠子对于你的用处，别废话了，我又不是吓大的，直接来谈条件吧！”
<br/>    
<br/>        说实话，若不是凌仙现在这小身子骨受不了这风沙，陈明还真是想直接把凌仙丢到风妖老祖的脸上，用弱智光环糊他一脸！
<br/>    
<br/>        风妖老祖思考了一会儿，然后道：“好，既然你知道我根本无法离开这个地方，那就说说你的条件吧。”
<br/>    
<br/>        陈明看向风妖老祖，然后道：“这样，我将这一颗内丹给你，你俯首称臣，如何？”
<br/>    
<br/>        风妖老祖哈哈大笑，道：“你这个人类，也不要耍那些小心思了，就是曾经抓住我的御剑宗，也没有能够让我臣服，只是让我护佑他的山门，你何德何能？”
