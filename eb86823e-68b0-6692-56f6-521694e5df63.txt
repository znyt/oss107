　　第718章
<br/>　　没有名份二字，想要钱通玉三家势力心服口服是件极不容易的事情。只要李乐天一道诏书，自己就是天下皆可讨之的山贼。他们三家不会愿意随着自己做山贼。
<br/>　　同样，也只需要李乐天一道诏书，自己就能名正言顺。让他们三家归附信任凉州王，便也不是很难克服心理障碍。
<br/>　　如此看来，长安是非去不可。
<br/>　　刘悲已劝过程大雷一次，见此刻程大雷脸上表情，便已知道他打定注意。
<br/>　　“大当家要去的话，也不是不行，但需做好准备，这次去长安，必定危机重重。”
<br/>　　“怕什么，关关难过关关过，咱们能走到今天，也不完全是因为运气。”程大雷目光转了一圈：“怎样，谁想陪我去长安凑凑热闹？”
<br/>　　刘悲是闲散性子，对于这种凑脑袋的事情实在不感兴趣。吴用有些跃跃欲试，恨不得毛遂自荐。
<br/>　　程大雷目光越过他，落在徐神机身上：“徐军师，你是逃不过的？”
<br/>　　“呃，为何次次都是我。”徐神机知道这次行动的危险程度，况且旅途劳顿，那如在凉州享清福。
<br/>　　程大雷没有回应他，心里琢磨着这次行动。
<br/>　　既然要去，便必须早做准备，距离八月十五还有些时间，但从凉州到长安，路上也需要一些时间，如此算起来，其实时间并不宽绰。
<br/>　　另外，与钱通玉三家的谈判也在进行中，这件事由刘悲与吴用负责。据刘悲所说，接下来的谈判极其顺利，他们三家已答应派人质到凉州城。
<br/>　　尤其是孙别山，还愿意将自己妹妹嫁给程大雷，做妾也是无妨。从刘悲口中得知，那名叫孙卸珠的姑娘天生好相貌，生得虎背熊腰，双臂有千钧之力。座骑五花马，披烂银铠，弯弓射箭，百发百中。
<br/>　　刘悲和吴用都有意促成这件事，为了山寨，为了大家，为了大家的山寨，大当家就该忍辱负重，以八尺之躯，承受千钧之重。
<br/>　　在程大雷将他们痛骂一番后，二人才偃旗息鼓，不再提这件事。
<br/>　　程大雷余怒未消，心里想这孙别山到底是直性子，还是特意恶心自己。连基本的审美观都缺乏么？
<br/>　　无论如何，这件事算是谈妥了。三人分别差人将人质送到凉州城，入城的第一件事就是来见程大雷。
<br/>　　钱通玉送来的是他的三公子，一个九岁男孩，小名唤作钱元宝。
<br/>　　孙别山膝下无子，送来的是他的女儿，唤作孙盈盈，今年方才十一岁。
<br/>　　白空集送来的是他侄子，名唤白画山，今年一十六岁，已经是一把推不倒的后生。
<br/>　　看三人送来的人质，便知道三人的态度。这白空集终究是不肯完全信任自己，真如果有朝一日双方动起手来，侄子这种存在，牺牲了也就牺牲了。
<br/>　　如今，三人都立在程大雷面前，看脸上的表情都有些忐忑，毕竟，程大雷的名声不是如何好。这坏人会不会把自己剁碎了做成下酒菜。
<br/>　　“在家里都读过书么？”程大雷问。
<br/>　　“禀程当家，有先生教过识字，不敢说过读书。”
<br/>　　倒是钱元宝这人激灵些，率先开口回答程大雷的问话。其他两人都战战兢兢，看来很怕程大雷。
<br/>　　程大雷撇撇嘴，心道本当家一向随和可亲，谁人不知谁人不晓。
<br/>　　“习过武么？”
<br/>　　小元宝摇了摇头，孙盈盈自然不会去学男人打打杀杀，只有白画山老老实实道：“跟府中的教师习过枪棒。”
<br/>　　程大雷点点头：“文韬武略，文韬武略，逢着这样的世道，文韬武略总有用得着的地方，有你们出头的机会。”
<br/>　　程大雷吩咐人安排他们的住处，人质过的是寄人篱下的日子，虽然说衣食无缺，但难免会有些冷嘲热讽。
<br/>　　不过，程大雷没打算虐待他们，自然会为他们安排好一切。再者说，他们这次过来，也不会是独身一人，随行的丫鬟仆役，马夫花匠都有十几人。
<br/>　　这些人一起都安排下，在将他们三个安排妥当后，江山图上，浩甲、武威、金城三座城池被点亮。
<br/>　　凉州城、朔方、蛤蟆城、武威、浩甲、金城连在一起，抱成一团，程大雷事实上已经算是掌握整个凉州。
<br/>　　差的，就是一个名份了。
<br/>　　于此同时，程大雷也收到了完成任务的系统提示。
<br/>　　嘟，恭喜完成任务，获得将军令一枚。
<br/>　　程大雷很快就在书房内找到这枚系统奖励的将军令，不出意外，上面纹着一只癞蛤蟆。他将崔白玉几人关在门外，自己躲在屋里查看这枚将军令。
<br/>　　自己现在是凉州之主，怎么着也该自己个藩王令，只是一枚将军令，系统是不是有些不大看得起自己。
<br/>　　这枚将军令的作用是提升佩戴者的魅力值，另外的作用是查看某人的忠诚度，一天只能对一人使用。
<br/>　　程大雷迫不及待想试一试这令牌的作用，走出书房面对在外面处理书卷的崔白玉。
<br/>　　眼睛盯着崔白玉，脑海中便浮现崔白玉的信息，最下面一行多了一条。
<br/>　　忠诚度：零。
<br/>　　程大雷觉得自己浪费了这次机会，想也知道，崔白玉不会对自己产生忠诚这种东西。这枚将军令无法显示负值，否则会显示崔白玉对自己恨之入骨。
<br/>　　还是要给这姑娘刷一刷好感度，程大雷打开首领光环，理了理衣服。
<br/>　　“怎样，在忙什么？”
<br/>　　崔白玉想要站起，程大雷冲她摆摆手，示意她坐着说。崔白玉正帮着刘悲募兵，一些军饷粮草的事情。
<br/>　　程大雷亲手掌握的兵力不过三万余，这些兵力控制整个凉州其实是不够的。
<br/>　　“不必太忙碌，平常也要想着歇一歇么，不要把自己累坏了。”
<br/>　　崔白玉一头雾水，程大雷这种关切的话语，也令她觉得毛骨悚然。
<br/>　　“其实有件事要同你商量。”
<br/>　　“大当家吩咐便是。”
<br/>　　“你想不想回家看看？”
<br/>　　崔白玉一楞，抬起头难以置信的盯着程大雷。纵然黑纱遮面，这美丽也令人无法直视。
<br/>　　崔白玉来自相府，所谓回家，便是回长安。
<br/>　　“想不想去长安走一走？”程大雷确定道。
<br/>(本章完)
