　　黑云学教一众弟子，虽吐出了‘出’字，但脸色，同样复杂。
<br/>　　因为他们这个曾经敬爱的师兄，敢拿他们的命去拼，去赌。
<br/>　　但在想到他妹妹时，却毫不犹豫放弃了一切拼、赌的念头。
<br/>　　萧逸，本脸色冰冷。
<br/>　　但，最后化作了轻淡。
<br/>　　他本对这个当年在黑云学教心存几分尊重的师兄，多少留有一点希冀。
<br/>　　今日，若莫悠肯点头，他萧逸也会毫不犹豫与他联手，与这古帝主局拼个高低。
<br/>　　可，莫悠的摇头，已然让萧逸明白，这，就是莫悠。
<br/>　　他根本连半分希冀都不该存。
<br/>　　这，对他而言，只会是个永远的陌路人。
<br/>　　故他甚至连淡漠的脸色都不复，只是化作了轻淡。
<br/>　　“莫悠，你…”金尘脸色一变，随后一冷。
<br/>　　原本他想说些什么，但他又反应过来，现在，无论说什么都无用了。
<br/>　　莫悠以及黑云学教一行人，已然说出了‘出’字。
<br/>　　洞府禁制，已然生效。
<br/>　　顶多十数分钟之后，一行人就会自动离开古帝洞府。
<br/>　　恰在此时。
<br/>　　嗖…嗖…嗖…
<br/>　　一众一流势力天骄，悉数凭空消失于原地。
<br/>　　在之前，萧逸一行人入了黑云大阵后，五大学宫对这些天骄一流天骄的掣肘就已然停止。
<br/>　　而那时，一众一流天骄，就已然喊出了‘出’字。
<br/>　　不过在主局之内，外头的洞府禁制效果极差，这才一直到现今才在洞府禁制之下消失离去。
<br/>　　对这些一流天骄来说，可是半刻也不想在这危险的主局内多留。
<br/>　　现今，还在主局之内的，便只剩一位位带队者，还有贺幽、血柏等十八府天骄。
<br/>　　“呼。”顾非凡深呼吸一口气，摇了摇头，“我们也走吧。”
<br/>　　“出。”
<br/>　　顾非凡率先说了一声，一众学宫弟子点了点头，也只能吐出‘出’字。
<br/>　　其余四大学宫弟子，也只能摇了摇头，纷纷吐出‘出’字。
<br/>　　令狐忘、金尘等人，脸上写满了不甘，以及愤懑。
<br/>　　但莫悠和黑云学教一行人即将离去，已是事实。
<br/>　　单凭他们五大学宫，根本没可能破局。
<br/>　　故，他们也只能选择离去。
<br/>　　而另一边，凌鸿咬了咬牙，脸上，同样写满了不甘，“该死。”
<br/>　　“唉。”贺幽叹了口气。
<br/>　　“算了吧，莫悠和黑云学教离去，五大学宫也已放弃。”
<br/>　　“单凭我们，没机会的。”
<br/>　　贺幽、血柏、七星府少府主、撼星府少府主等一位位少府主，悉数喊出了‘出’字。
<br/>　　凌鸿，沉思了半分，也选择了离去。
<br/>　　最后，是傲吟风，叶流、冉琦、第一云、长孙赤烈、花若怜，顾怜星，以及公孙火舞。
<br/>　　此时，傲吟风看了紫电剑阵几眼，随后看向萧逸，皱眉道，“萧逸副殿主，若在下没看错的话，你这剑阵之内，有我东离剑宫的无上剑诀意味在。”
<br/>　　傲吟风，作为东离剑宫第一天骄，自是一眼就能认出无上剑诀的意味。
<br/>　　不错，紫电剑阵之内，萧逸确实夹杂了无上剑诀的剑气。
<br/>　　事实上，紫电确实厉害。
<br/>　　但以萧逸现今修为，根本不可能将紫电这把上品巅峰圣器里头的所有威力悉数发挥出。
<br/>　　当然了，无论他能发挥出多少，那些紫电之力，都在他绝对掌控之下。
<br/>　　而在这之前，他守着剑道巨门之时，可不是傻乎乎在白白浪费时间。
<br/>　　他一边抵挡妖兽之时，也一边尝试着无上剑诀。
<br/>　　无上剑诀，之所以有‘无上’二字，便是因为它能增幅在任何剑道和剑道手段之中。
<br/>　　而那时，他也发现了无上剑诀的另一个特点。
<br/>　　那便是越狂猛的剑道和剑气，其增幅就越大。
<br/>　　紫电剑气，毫无疑问狂暴到极点，加诸无上剑诀后，更是增幅极大。
<br/>　　故萧逸之前拿寻常的剑道，还有星幻剑道以及剑气去尝试让无上剑诀增幅，收效甚微，效果也不大。
<br/>　　反倒在以紫电剑气与之契合后，萧逸参悟效果大升，收获也极大。
<br/>　　借紫电之内雷电之力，外加无上剑诀的增幅，方才有了紫电剑阵地域如潮妖兽。
<br/>　　甚至是莫悠的两只绝世9999道傀儡在其中也无法支撑多久。
<br/>　　此时，听闻傲吟风的话语，萧逸点了点头，道，“不错。”
<br/>　　“我那位易兄，并不修剑道，故他把无上剑诀给了我。”
<br/>　　“我练了练，效果倒是不错。”
<br/>　　傲吟风仍旧皱着眉，“原本，你在剑域杀我剑宫弟子，你我乃是大仇。”
<br/>　　“不过，易霄副殿主与我东离剑宫以及宫主，交情颇深。”
<br/>　　“看在这份上，萧逸副殿主与我东离剑宫之恩怨，便一笔勾销吧。”
<br/>　　傲吟风边说着，拱了拱手。
<br/>　　“哼。”一旁公孙火舞冷哼一声。
<br/>　　“说得倒是好听，凭今日双生子之名，你东离剑宫敢说仇怨二字？”
<br/>　　“以你的实力，又能奈何萧逸？”
<br/>　　顾怜星嗤笑一声，“据我所知，当初在剑域之内，你东离剑宫弟子，可是打算抢夺剑帝本源，与天下剑修联手对付萧逸公子。”
<br/>　　“这才逼得萧逸公子大杀四方。”
<br/>　　傲吟风皱了皱眉，“便是我也知晓此事，方才与萧逸副殿主握手言和。”
<br/>　　“否则，若是萧逸副殿主当真故意对我剑宫弟子下毒手，无论他是何身份，我傲吟风绝不会罢休。”
<br/>　　“罢了。”萧逸摇了摇头，语气轻淡。
<br/>　　事实上，也没什么好说的。
<br/>　　当初那批剑宫弟子利欲昏心，为夺机缘，不惜联手也要杀人夺机缘，那就要有被反杀的觉悟。
<br/>　　萧逸杀了他们，也是情理之中。
<br/>　　而对于东离剑宫，萧逸也并不怪。
<br/>　　毕竟一码归一码。
<br/>　　“无论如何。”傲吟风拱了拱手，“此次承蒙萧逸副殿主一直护着，吟风才有命离开这主局。”
<br/>　　“谢谢。”
<br/>　　傲吟风答谢一声。
<br/>　　萧逸点点头，不语。
<br/>　　这时，众人都已然说出了‘出’字。
<br/>　　不过，众人却也发现，全场，有一人没有说。
<br/>　　正是萧逸。
<br/>　　“萧逸兄弟，你不打算走？”叶流率先反应了过来。
<br/>　　“你没喊出字？”冉琦微微有些后知后觉。
<br/>　　“萧逸，你…”第一云等人，面露惊色。
<br/>　　“萧逸师弟。”莫悠皱着眉。
<br/>　　“萧逸，你别乱来。”青麟、童烨一行人，认真地看着萧逸。
<br/>　　“呵。”萧逸淡淡一笑，“我暂时不走。”
<br/>　　“我的命，我想赌一把。”
