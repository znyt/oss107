　　第99章
<br/>　　若论马战，戎族骑术天下无双。
<br/>　　三十年前，戎族浩荡而来，杀人如割草，破城如踏灰。偌大一个帝国，十三州一百单八城无人能挡。
<br/>　　后来戎族被赶跑了，说是赶跑，但其实具体怎么会事，大家心里都清楚。
<br/>　　这件事过去后，帝国的军人就患上了恐戎症，一听戎族两个字就打哆嗦，还有什么『戎族不满万，满万不可敌』的说法跑出来。
<br/>　　黑石城便是帝国的边界，是极北之地的最后一城。陆哼虽没有和呼尔勒照过面，关于他的名声却是听说过的。
<br/>　　所谓闻名不如见面，见面更胜闻名。堂堂戎族小王子亲自上阵，连杀十一人，尸体倒在地上可以连成一条线。
<br/>　　人群慌乱，那些慷慨热血的才子面若筛糠，身体在马上、驴上、牛上瑟瑟发抖。
<br/>　　热血有用，也有限。
<br/>　　这一回，陆哼亲自上阵。
<br/>　　拍马疾驰，借着马力手中铁棍狠狠扎向呼尔勒。呼尔勒咧嘴而笑，见陆哼逼得近，身子一下藏在马肚子下。
<br/>　　陆哼再回过神来，眼前已经无人，忽然脑后风声紧，一道杀意袭来。陆哼整个人一激灵，反身将铁棍挡在身后。
<br/>　　Duang！
<br/>　　呼尔勒的铁斧已经劈了过来，正好砸在铁棍上，陆哼如撞到铁牛上，五脏六腑似乎都被震碎，喉头一甜，脑袋晕晕乎乎，身体跌落在马下。
<br/>　　真若说陆哼在呼尔勒手下走了两个回合，实在是抬举陆哼。其实他一个回合都没能完成，双方的实力差着一个量级。
<br/>　　这伙恶贼竟然这么强！
<br/>　　乡勇军内部已经军心涣散，到现在，他们还以为这帮人是蛤蟆寨一伙。
<br/>　　呼尔勒横斧立马，看陆哼如看猪狗，手中大斧举起，照着陆哼脑袋就劈了下来。
<br/>　　“死吧！”
<br/>　　陆哼的求生欲还是很强的，但此刻已走到绝路，只能眼睁睁看着那斧头从天空中落下。
<br/>　　终还要死在这斧头下啊。
<br/>　　“呼尔勒！”
<br/>　　恰在千钧一发之时，这声音忽然响起，循声音望过去，一行六人行到两军阵前。
<br/>　　呼尔勒一楞，手中的斧头停在空中：此地还有人知道我的名字。
<br/>　　这一行六人正是程大雷一伙，他们来得稍慢些。他们随着乡勇军一起出城，双脚步行速度自然快不了，也就是半路上抢了六匹马，才一路急匆匆赶来。
<br/>　　呼尔勒一伙看到气喘吁吁的程大雷就笑出声来。
<br/>　　“他们不就是那伙山贼么！”
<br/>　　“怎么，是又给我们送马来了！”
<br/>　　“你那枪不错，俺收下了。”
<br/>　　一伙人骑在马上，以睥睨的目光看着程大雷一伙，纷纷都咧嘴笑着。
<br/>　　山贼虽然不入流，也经常被人看不起，可无论官匪，看到自己后谁不唤一声『程当家』。这伙戎族，是明显看不起自己呐。
<br/>　　呼尔勒骑在马上，手持大斧，耀武扬威不可一世。
<br/>　　我的马，我的斧头！
<br/>　　程大雷将牙咬得咯吱咯吱响，如果我的『雄辩』技能修到MAX，一定用嘴炮杀得你口吐鲜血。
<br/>　　但如今，这技能还起不到用处，程大雷也只能放大招。
<br/>　　“秦蛮，杀人！”
<br/>　　秦蛮早在等程大雷这句话，程大雷憋了一肚子火，秦蛮何尝不是。他的铁枪如今还在赤虎手里握着呢，现在秦蛮只能用一根顶门杠当兵器。
<br/>　　秦蛮来势汹汹，一棍横扫，呼尔勒吃了一惊，帝国竟也有如此高手。他一个矮身躲过，双方战在一起，打得难解难分。
<br/>　　秦蛮打出火气，一棍向呼尔勒脑袋扫来，呼尔勒也抡起斧头。只听咔嚓一声，秦蛮手中的顶门杠只剩下半截。
<br/>　　呼尔勒哈哈大笑：“让你知道我风暴战斧的厉害！”
<br/>　　程大雷气得把牙都快咬碎了：“我的斧头！”
<br/>　　“秦当家，接兵器！”
<br/>　　陆哼此刻已站起来，手中的铁棍丢了过来。秦蛮凌空接住，以棍使枪，在空中挽了个枪花，奔着呼尔勒的咽喉而来。
<br/>　　呼尔勒的情况有些捉急，深吸一口气，把看家功夫也使了出来。只用双腿控马，手中的斧头抡圆了，赫赫杀威淋漓。
<br/>　　陆哼倒抽一口凉气：“此人的风暴战斧当真是恐怖如斯！”
<br/>　　“我的斧头！”程大雷再次咬牙。
<br/>　　程大雷看着二人的战斗，发出和陆哼一样的感慨：此人好俊的骑术。
<br/>　　他已经看出来，秦蛮不是此人对手。若论气力，二人旗鼓相当，都是顶级人物，可论骑术秦蛮不如呼尔勒。
<br/>　　再打下去，秦蛮赢不了。
<br/>　　“秦蛮，退！”
<br/>　　秦蛮虽然听程大雷的命令退下，可心里仍旧窝着一团火。
<br/>　　呼尔勒又胜一场，跨在马上更加得意：“我自从得到这柄斧头以来，手上还未尝败绩。”
<br/>　　张肥气哇哇大叫：“狗奴休要猖狂，燕人张肥在此！”
<br/>　　程大雷赶忙拦住他：“三爷，息怒，息怒，我知道十年后你杀他如杀狗，但现在你还真不行。”
<br/>　　“那如今怎么办？”徐神机问。秦蛮不行，林少羽就更够呛，现在谁还能迎战呼尔勒。
<br/>　　“我……亲自会他。”程大雷吐音如刀。
<br/>　　“大当家，慎重，慎重！”
<br/>　　程大雷乘马来到呼尔勒对面，剑仍在鞘中，并未出剑。
<br/>　　“我有一招，从未用过，汝能接否？”
<br/>　　呼尔勒冷笑：“装神弄鬼，你有什么能耐？”
<br/>　　“我问你能接否！”程大雷猛喝如雷。
<br/>　　呼尔勒一楞，在一刹那被程大雷气势所夺：“你……你想干什么？”
<br/>　　“能……接……否？”程大雷缓缓拔剑，剑与鞘摩擦，发出刺耳的响声。
<br/>　　“你有什么本事！”
<br/>　　“吾只有一招……”程大雷剑指呼尔勒，口中缓缓吐出两个字，字字如雷音。
<br/>　　“抽……奖”
<br/>　　脑海中的扭蛋机出现，骨碌碌旋转，程大雷求爷爷告奶奶，系统大大：送我一员顶级武将吧，加上秦蛮，以二打一，呼尔勒必死无疑。
<br/>　　一物从扭蛋机中掉下来，系统提示恰到好处的响起。
<br/>　　嘟，正义的少年喔，恭喜你获得传说级武学一册：三手矛（名将张翼德所传）
<br/>　　程大雷看了张肥一眼，这秘籍好是好，但委实派不上用场啊。
<br/>　　呼尔勒刚才被程大雷装神弄鬼搞得有点懵，他见程大雷举着剑半晌不动，只是手臂不停颤抖。
<br/>　　“你干嘛呢？”
<br/>　　“活跃一下气氛，看刚才多紧张啊，哈哈。”程大雷擦擦汗，用认真的眼神望着对方：“你信么？”
<br/>　　
<br/>(本章完)
