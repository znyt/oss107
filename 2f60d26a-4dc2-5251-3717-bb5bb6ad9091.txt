　　叶天歌听到这话，眼眸中顿时露出一抹担忧之色，不过很快便掩饰了过去，微笑道：
<br/>　　“你起的名字，自然是极好的！”
<br/>　　说完出这话的同时，叶天歌伸手摸了摸叶尘的脑袋，眼中透着浓浓的爱意。
<br/>　　前世，自叶尘记事起，就从未得到过父爱，即便后来知道自己的父亲是叶天歌，对他也只有恨意，还从未有过这种感觉。
<br/>　　眼下他们一家三口在一起，可以说温馨极了，甚至叶尘都恨不得永远停在这一刻。
<br/>　　然而，
<br/>　　事情总是事与愿违。
<br/>　　就在他们一家人，正其乐融融的时候，
<br/>　　砰！
<br/>　　房门突然被人从外面一脚踹开！
<br/>　　紧接着，
<br/>　　一群人从外面直接冲了进来，为首的是一名二十来岁的女子，论姿色不再林婉容之下，而且身上散发着一股凌厉的气息，虽然在叶尘看来根本不值一提，但是在地球上，应该也算是高手之列了。
<br/>　　“秦月娥？”
<br/>　　叶尘很快便认出了这个女人，这赫然正是叶无伤的母亲，当年将叶尘的母亲林婉容赶出京都的秦月娥！
<br/>　　“好你个下贱的女人！竟然连孩子都生出来！”
<br/>　　秦月娥闯进来之后，目光先是在林婉容的身上扫过，继而落在叶尘的身上，眼眸中透着浓浓的杀意。
<br/>　　“来人！给我把这个下贱女人生的野种杀了！”
<br/>　　“是！小姐！”
<br/>　　秦月娥一声令下，身后顿时窜出一名中年壮汉，直奔床上的叶尘冲了过来。
<br/>　　“握草！”
<br/>　　叶尘不由得暗暗爆了一声粗口，
<br/>　　眼前这个中年壮汉，在他眼中简直就是蝼蚁，也就比普通人强上一些罢了，可是他如今还是一个婴儿，连说都还不会说呢，哪里有力气反抗？
<br/>　　“放肆！我叶天歌的儿子，也是你能碰的！”
<br/>　　没等那中年壮汉冲上来，一旁的叶天歌，直接一声暴喝，同时一掌拍了出去！
<br/>　　那中年壮汉大约也就是暗劲武者的水准，连化劲宗师都还不是，而叶天歌此时虽然武功还未大成，还不是那个华夏国大名鼎鼎的军中战神，但对付眼前这个人，已经绰绰有余。
<br/>　　砰！
<br/>　　那中年壮汉还未踏入床前三米，就被叶天歌一道气劲直接拍飞了出去，狂喷了一大口鲜血，足足飞出十多米，方才重重坠落，然后头一歪，晕厥了过去。
<br/>　　蹬蹬蹬！
<br/>　　秦月娥的其他几名随从见状，顿时吓得纷纷倒退。
<br/>　　秦月娥脸色发青，眼眸中透着浓浓的妒火，
<br/>　　“叶天歌！你不要忘了！我才是你的原配夫人，如今你竟然和别人女人生下一个野种，还如此袒护他，真当我秦家是好欺负的吗？”
<br/>　　听到秦月娥将秦家搬了出来，叶天歌的脸上微微一变，声音发寒道：
<br/>　　“秦月娥，你明知道我根本不爱你，我此生唯一爱的人，只有容容！你为何要这么一而再、再而三的逼我？”
<br/>　　“你先是设计，让自己怀上了我的孩子，然后又用秦家的势力逼迫叶家，让我娶你为妻，现在竟然还想赶尽杀绝，连我和容的孩子都不放过，难道这样做，你就开心了吗？”
<br/>　　秦月娥冷冷一笑，目光转移到林婉容身上，眼眸中透着浓浓的嫉恨之色，几乎都要喷出火来了！
<br/>　　“论出身，论容貌，论武功，这个下贱的女人，哪一点比我强？你居然偏偏希望这个贱女人，我就是不服！若是我得不到的东西，别人也休想得到！”
<br/>　　秦月娥说到这里，微微一顿，目光又转移到叶尘的身上，寒声道：
<br/>　　“之前你说过，只要我不找这个贱女人的麻烦，你就答应和我成亲，我秦月娥说到做到，绝对不会动她一根手指头，可是这个贱女人生的野种，我今天必须要杀！”
<br/>　　轰！
<br/>　　秦月娥说完这话的同时，突然身躯一晃，闪电般冲到床前，直接一掌便拍落了下来！
<br/>　　叶尘顿时两眼一瞪，虽然以他的眼界，对方的实力根本不值一提，可是如今的他，哪里有能力抵挡？
<br/>　　好在叶天歌反应极快，没等秦月娥这一掌拍下，已经抢先一步拦住，同时猛然一震，
<br/>　　蹬蹬蹬！
<br/>　　秦月娥顿时一连倒退了七八步，方才勉强站稳身形。
<br/>　　林婉容反应过来之后，顿时惊呼了一声，慌忙一把将叶尘紧紧搂在怀里，竭力呼喊道：
<br/>　　“你要杀就杀我好了，不要动我的孩子！”
<br/>　　叶天歌脸色阴沉无比，死死盯着秦月娥，眼眸中开始浮现出一抹杀意，
<br/>　　“秦月娥！你一而再的挑衅我的极限，难道真的以为我不敢杀你吗？”
<br/>　　轰隆！
<br/>　　叶天歌声音落下，身上升起一股极强的煞气，显然已经对秦月娥起了杀心。
<br/>　　咚！咚！咚！
<br/>　　叶天歌缓步朝角落里的秦月娥逼近，每迈出一步，地面就轰轰作响，犹如地震了一般。
<br/>　　秦月娥顿时脸色大变，不由得连连倒退，满脸惊恐之色，
<br/>　　“天歌，你、你想干什么？”
<br/>　　叶尘看到这幅场景，心里也不由得呐喊起来，
<br/>　　“杀了她！快点杀了她！”
<br/>　　如果此时他有能力的话，肯定毫不犹豫的把这个可恶的女人弄死了。
<br/>　　然而就在这时，
<br/>　　忽然一道苍老的声音，陡然从屋外响起，
<br/>　　“我倒要看看，谁敢动我秦踏天的孙女！”
<br/>　　声音落下的同时，房间突然多出一道老者的身影。
<br/>　　只见这人一身紫袍，须发皆白，气势逼人，赫然正是秦家的家主秦踏天！
<br/>　　看到眼前的这些人，叶尘忽然有些分不清，自己眼前所在的世界，究竟是另外一个时空，又或者只是根据自己的记忆，而创造的一个无比逼真的幻境了？
<br/>　　不过此时，已经容不得他去多想，秦踏天到来之后，叶天歌顿时脸色大变，气势瞬间就被对方压制了下去。
<br/>　　而那秦月娥则满脸喜色，快速跑到秦踏天的跟前，
<br/>　　“爷爷，你要为我做主啊！这个贱女人，竟然连孩子都生下来了，我可咽不下这口气！”
<br/>　　“哼！”
<br/>　　秦踏天冷哼一声，脸上露出一抹不耐之色，
<br/>　　“既然这男人如此便将你放在心上，那爷爷把他们全部灭了就是，顺便将叶家也连根拔起，岂不干净痛快？”
