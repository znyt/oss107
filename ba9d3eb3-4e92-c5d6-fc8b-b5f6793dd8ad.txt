　　第736章
<br/>　　暗道内伸手不见五指，空气潮湿，环境逼仄，是鼠虫的乐土。行走其中，不时会碰见一两只干瘦的老鼠，发出吱呀乱叫。
<br/>　　程大雷走在前面，刘发财断后，在他面前依次是香火僧慧禅，徐神机，鱼洞真与崔白玉和乔弄云。
<br/>　　几人也只能硬着头皮往前走，便是崔白玉和乔弄云也没有对周围的环境大惊小怪，毕竟逃命要紧，也顾不得周边有没有老鼠。
<br/>　　短短三里路，走了半个时辰，出口是一个山洞。程大雷踏出山洞，长长出了口气，由此望过去，汝南城尽收眼底。
<br/>　　“我就说么，小小一座汝南城困不住我，这不就出来了。”程大雷把额头的汗珠抹去。
<br/>　　“但你还是要回去？”崔白玉问。
<br/>　　“嗯呐，把那憨货接出来。”程大雷。
<br/>　　崔白玉有时感觉自己永远不会搞懂程大雷。在汝南城一通乱杀，若是识趣人有机会逃生的话，早就马不停蹄的离开这是非之地。可程大雷为了一头畜生，偏偏却要选择折回。当然，他对一个牲畜尚且如此，对人就更不用说了，这样一个老大，也值得人为他卖命。
<br/>　　如此，也不好说他是聪明或者傻。
<br/>　　“大当家，我们在哪里等你？”徐神机。
<br/>　　“我已经问过了，从这里往北继续走十里，有一个秋风渡，你们在哪里等我，最多三天，三天后我与你们汇合。”程大雷。
<br/>　　“假若三天后你出不来呢，汝南有五千城防兵，也不是你想走就能走的。”崔白玉。
<br/>　　“怎么，你担心我？”程大雷看了她一眼：“如果三天内没见到我，你们就回凉州，之后再做打算。”
<br/>　　说罢，程大雷慧禅重新走回山洞，沿着来时的路返回清风寺。
<br/>　　于此同时，清风寺内，了然和尚正在进行天人交战。
<br/>　　他令弟子围住那头凶恶的黑牛，十几个身强力壮的和尚手里拎着钢刀面有难色。
<br/>　　佛家戒杀生，主持却让自己对一头无辜的畜生下手。
<br/>　　了然和尚有他自己的想法，如今汝南城危机四伏，程大雷就该早早逃命，但也不知道他抽了哪门子疯，围了一头畜生竟然冒险留在此地。
<br/>　　程大雷不能死，程大雷死了崔家就没了死灰复燃的希望。了然和尚思来想去，干脆将这头黑牛宰了，断了程大雷的念想，令他不想走也得走。
<br/>　　做出这个决定后，如何执行却成了一个问题。程大雷这头黑牛实在太过丑陋，看着让人怕，而且是杀生，整个清风寺竟然无一人敢下手。
<br/>　　如果不是年老体衰，了然和尚就要自己动手，可他实在有这个心无这个力。
<br/>　　“怕什么，无非是一头畜生而已。慧来，你去动手。”
<br/>　　名叫慧来的和尚咧咧嘴：“主持师傅，这可是杀生。”
<br/>　　“什么杀生，佛门弟子就要有我不下地狱谁下地狱的胸怀，杀一救万扛下所有罪责，这才是大境界。”
<br/>　　慧来虽然仍拿着刀，但并没有被忽悠住，仍然不敢上前动手。
<br/>　　那头黑牛被围在当中，睁大一双大眼，却也不明白他们在做什么。只是百无聊赖的咀嚼着干草。
<br/>　　“喂，法师你们干什么呢？”
<br/>　　程大雷不知什么时候已返回。了然吓了一跳，忙道：“没什么，没什么，我看程施主这头坐骑有些凶恶，怕是有些来历。”
<br/>　　“嘿，让你说着了。不是本当家夸口，天上地下怕就是这独一份，好几次我是仗着它才能脱险。”
<br/>　　程大雷能为一头畜生留下，如果知道自己要宰了他的坐骑，还不知会发怎样的脾气。了然别无他法，只能把佛家不打诳语那一套丢在脑后，捧着程大雷聊。
<br/>　　见把话题转开，他方才道：“程当家可有脱身的法子？”
<br/>　　“暂时还没有。”
<br/>　　“呃……”
<br/>　　“怕什么，办法总会想出来的，只要给我一条路，我就有办法冲出去。”
<br/>　　摆脱崔白玉几个累赘后，程大雷信心大增，也不像刚开始那样紧张。
<br/>　　既然敢出凉州，他就预见到会有这样一幕。实在不行，自己动用系统召唤几员大将出来，好好闹一闹这汝南城。
<br/>　　程大雷透过清风寺的山门向外张望一眼，见汝南城的官兵依旧围着水泄不通，他们虽然没有进攻，但也并没有撤退。
<br/>　　却也不知他们打算守到什么时候，程大雷忽地打开山门，从山下喊了一嗓子。
<br/>　　“呔，吾乃凉州程大雷，奉陛下之命前往长安，你们胆敢阻拦我的路，想要造反不成。”
<br/>　　负责指挥的将官吓了一跳，程大雷这厮胆子好大，都到现在了都敢叫阵。
<br/>　　“你们与我听仔细了，识趣的快快撤开，否则我到了长安，启禀陛下后先定你们一个造反之罪，杀了你们的脑袋，你们也是反贼。”
<br/>　　程大雷这话的确起到了一定作用，对于底层兵丁来说，死不可怕，怕的是死了也得给祖坟抹黑。
<br/>　　魏牧冲也在军中，他知道程大雷这话的攻心作用。他本人不在意天怒人怨，心里也明白，即便陛下乐意程大雷去死，杀掉他后陛下也会治自己的罪。他早已将生死置之度外，又岂会怕程大雷这三言两语。
<br/>　　说来说去，凉州是块烂疮，容他存在，危害只会越来越大，就该及早剜掉。除了程大雷，凉州群龙无首，即便打过来，京州也未必守不住。
<br/>　　相比之下，他的目光要看得更长远一些。
<br/>　　“程贼，你莫要妖言惑众，陛下早已有密旨给我，就是要在此地除掉你。今日谁能杀掉此贼，生官一级，俸银加倍。”
<br/>　　对于程大雷的攻心之策，魏牧冲也开始忽悠。
<br/>　　“呵呵，姓魏的，你说的话有谁会信。你是个连自己儿子都敢杀的人，你想要大义灭亲的名声，可全天下都会说你丧尽天良。来来来，让本当家趁早砍掉你的脑袋，你去地府赎罪去吧。”
<br/>　　二人一个山下一个山上，魏牧冲做过御史，职业就是靠骂人吃饭。程大雷也是牙尖嘴利，两人一时之间竟分不出个胜负。
<br/>　　正这时候，从对方军中冲出一员小将，骑在马上大喝：“姓程的，快快快，来我马前送死。”
<br/>(本章完)
